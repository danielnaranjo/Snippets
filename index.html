Easy way to use share button and CSS Sprites

A little history..

Once I had to place some social buttons in a horrible design provided by a customer, I wanted to do something fast, functional and future use so I wrote this simple solution:

 

 

HTML

			<div id="social">

			<div class="facebook"		data-user="yourfacebook"></div>

			<div class="twitter"		data-user="yourtwitter"></div>

			<div class="youtube"		data-user="youryoutube"></div>

			<div class="gplus"		data-user="u/0/yournumber"></div>

			<div class="linkedin"		data-user="in/yourlinkedin"></div>

			</div>

 

CSS

#social {

			display: inline-block;

			height: auto;

			margin: 0;

			padding: 0;

			vertical-align: top;

			width: 100%;

			}

			.facebook,.twitter,.youtube,.linkedin,.gplus,.youtube {

			background-image: url("sociales.png");

			background-repeat: no-repeat;

			height: 34px;

			width: 32px;

			margin-right: -5px;

			display: inline-block;

			vertical-align: top;

			cursor: pointer;

			}

			.facebook {background-position: 0px 0px;}

			.twitter {background-position: -32px 0px;}

			.gplus {background-position: -92px 0px;}

			.youtube {background-position: -62px 0px;}

			.linkedin {background-position: -122px 0px;}
